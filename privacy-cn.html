📚 ReadLens 隐私政策

最后更新：2025年10月11日

🔒 核心承诺

ReadLens 是一个完全本地化的应用程序。您的所有数据都存储在您的设备上。我们绝不收集、存储或传输您的个人数据。

1. 我们收集哪些数据？

简短回答：无

ReadLens 不收集任何个人数据。所有数据都存储在您的设备本地。

详细说明

ReadLens 将以下数据仅存储在您的设备上：

  • 保存的文章：您选择保存的网页内容
  • AI 摘要：使用您的 API 密钥生成的摘要（如果启用）
  • 标签和笔记：您添加到文章的标签和个人笔记
  • 设置：您的应用偏好设置（主题、语言等）
  • API 密钥：您的 AI 服务 API 密钥（本地加密）

以上所有数据仅存储在您的设备上。ReadLens 团队无法访问这些数据。

2. 我们如何使用您的数据？

ReadLens 不使用或访问您的数据，因为：

  • ✅ 无服务器：没有后端服务器收集数据
  • ✅ 无账户系统：无需注册或登录
  • ✅ 无数据上传：数据永不离开您的设备
  • ✅ 无第三方跟踪：没有 Google Analytics 或任何跟踪工具

3. AI 功能和 API 密钥

🔑 BYOK（自带密钥）模式

ReadLens 使用"自带密钥"方式。这意味着：

您的 API 密钥

  • 加密存储：API 密钥使用 AES-256 加密存储在您的设备上
  • 直接连接：ReadLens 直接将您的内容发送到 AI 提供商（OpenAI、Claude 等）
  • 无中间人：我们没有中间服务器；我们无法看到或记录您的请求
  • 您的控制：您可以随时删除或更改 API 密钥

向 AI 提供商传输数据

当您使用 AI 摘要功能时：

  • 您的文章内容直接从您的浏览器发送到您选择的 AI 提供商
  • 这些数据不会通过 ReadLens 服务器（我们没有任何服务器）
  • 请查看各 AI 提供商的隐私政策：
    • OpenAI 隐私政策
    • Anthropic (Claude) 隐私政策
    • Google (Gemini) 隐私政策

4. Chrome 扩展权限

ReadLens 需要以下 Chrome 权限：

activeTab（当前标签页）

目的：读取您正在查看的网页内容以保存文章

数据去向：仅存储在您的本地设备

storage 和 unlimitedStorage

目的：在您的设备上存储文章、设置和 API 密钥

数据去向：仅存储在您的本地 Chrome 存储中

contextMenus（右键菜单）

目的：提供右键快速保存功能

数据去向：无数据收集

notifications

目的：显示保存成功/失败通知

数据去向：无数据收集

https://*/*（网络访问）

目的：从网页提取内容并连接到 AI 提供商 API

数据去向：仅发送到您选择的 AI 提供商（如果启用 AI 功能）

5. 第三方服务

ReadLens 本身不使用任何第三方服务。但是：

可选的 AI 服务（BYOK）

如果您选择启用 AI 摘要功能并提供您自己的 API 密钥：

  • 您的文章内容将发送到您选择的 AI 提供商
  • 这些提供商有自己的隐私政策
  • ReadLens 不会看到或记录这些传输

我们不使用的服务

  • ❌ Google Analytics
  • ❌ 广告网络
  • ❌ 社交媒体跟踪
  • ❌ 任何形式的使用跟踪

6. 数据安全

本地加密

  • API 密钥使用 AES-256 加密存储
  • 加密密钥存储在您的浏览器中
  • 即使有人访问您的设备存储，没有解密密钥也无法读取 API 密钥

输入验证

  • 所有用户输入都经过验证和清理
  • 防止 XSS（跨站脚本）攻击
  • HTML 内容自动清理

7. 数据保留和删除

数据保留

由于所有数据都存储在您的设备上：

  • 数据将保留直到您主动删除它
  • 您可以随时在设置中清除所有数据
  • 卸载扩展会自动删除所有 ReadLens 数据

如何删除数据

  1. 删除单篇文章：在文章列表中点击删除按钮
  2. 清除所有数据：设置 → 数据管理 → 删除所有数据
  3. 完全移除：从 Chrome 卸载 ReadLens 扩展

8. 儿童隐私

ReadLens 不针对13岁以下儿童。我们不会故意收集儿童的个人数据（事实上，我们不收集任何人的数据）。

9. 隐私政策变更

如果我们更改此隐私政策：

  • 我们会更新本页面顶部的"最后更新"日期
  • 重大变更将在扩展更新说明中公布
  • 您随时可以查看最新版本的隐私政策

10. 您的权利

由于 ReadLens 不收集任何数据，传统的数据主体权利（访问、更正、删除等）不适用。

但您拥有完全控制权：

  • ✅ 随时查看、编辑或删除您的本地数据
  • ✅ 随时导出您的数据（JSON、Markdown、CSV 格式）
  • ✅ 随时禁用或移除 AI 功能
  • ✅ 完全控制您的 API 密钥

12. 适用法律

本隐私政策受台湾法律管辖。对于欧盟用户，我们遵守 GDPR 原则（尽管由于我们不收集数据，大多数 GDPR 要求不适用）。

💡 总结

ReadLens 的隐私承诺很简单：我们看不到您的数据，因为它永远不会离开您的设备。

  • ✅ 零数据收集
  • ✅ 零跟踪
  • ✅ 零上传到我们的服务器（因为我们没有任何服务器）
  • ✅ 完全的隐私保护
