📚 ReadLens プライバシーポリシー

最終更新日：2025年10月11日

🔒 コアコミットメント

ReadLens は完全にローカルなアプリケーションです。すべてのデータはお客様のデバイスに保存されます。私たちはお客様の個人データを収集、保存、送信することは決してありません。

1. どのようなデータを収集しますか？

短い回答：何も収集しません

ReadLens はいかなる個人データも収集しません。すべてのデータはお客様のデバイスにローカルに保存されます。

詳細説明

ReadLens は以下のデータをお客様のデバイス上にのみ保存します：

  • 保存された記事：保存を選択したウェブコンテンツ
  • AI 要約：API キーで生成された要約（有効な場合）
  • タグとメモ：記事に追加したタグと個人メモ
  • 設定：アプリの環境設定（テーマ、言語など）
  • API キー：AI サービスの API キー（ローカルで暗号化）

上記のすべてのデータはお客様のデバイスにのみ保存されます。ReadLens チームはこれらにアクセスできません。

2. データをどのように使用しますか？

ReadLens はお客様のデータを使用またはアクセスしません。理由：

  • ✅ サーバーなし：データを収集するバックエンドサーバーはありません
  • ✅ アカウントシステムなし：登録やログインは不要です
  • ✅ データアップロードなし：データはデバイスから出ることはありません
  • ✅ サードパーティトラッキングなし：Google Analytics やトラッキングツールはありません

3. AI 機能と API キー

🔑 BYOK（Bring Your Own Key）モデル

ReadLens は「自分のキーを持ち込む」アプローチを使用します。これは次を意味します：

お客様の API キー

  • 暗号化ストレージ：API キーは AES-256 暗号化でデバイスに保存されます
  • 直接接続：ReadLens はコンテンツを AI プロバイダー（OpenAI、Claude など）に直接送信します
  • 仲介者なし：中間サーバーはありません。リクエストを見たり記録したりすることはできません
  • お客様の管理：いつでも API キーを削除または変更できます

AI プロバイダーへのデータ転送

AI 要約機能を使用する場合：

  • 記事のコンテンツは、ブラウザから選択した AI プロバイダーに直接送信されます
  • このデータは ReadLens サーバーを経由しません（サーバーはありません）
  • 各 AI プロバイダーのプライバシーポリシーを確認してください：
    • OpenAI プライバシーポリシー
    • Anthropic (Claude) プライバシーポリシー
    • Google (Gemini) プライバシーポリシー

4. Chrome 拡張機能の権限

ReadLens は以下の Chrome 権限を必要とします：

activeTab（現在のタブ）

目的：記事を保存するために表示中のウェブページからコンテンツを読み取ります

データの宛先：ローカルデバイスにのみ保存されます

storage と unlimitedStorage

目的：デバイスに記事、設定、API キーを保存します

データの宛先：ローカル Chrome ストレージにのみ保存されます

contextMenus（コンテキストメニュー）

目的：右クリックで素早く保存する機能を提供します

データの宛先：データ収集なし

notifications

目的：保存の成功/失敗通知を表示します

データの宛先：データ収集なし

https://*/*（ネットワークアクセス）

目的：ウェブページからコンテンツを抽出し、AI プロバイダー API に接続します

データの宛先：選択した AI プロバイダーにのみ送信されます（AI 機能が有効な場合）

5. サードパーティサービス

ReadLens 自体はいかなるサードパーティサービスも使用しません。ただし：

オプションの AI サービス（BYOK）

AI 要約機能を有効にして独自の API キーを提供する場合：

  • 記事のコンテンツは選択した AI プロバイダーに送信されます
  • これらのプロバイダーには独自のプライバシーポリシーがあります
  • ReadLens はこれらの送信を見たり記録したりすることはありません

使用していないサービス

  • ❌ Google Analytics
  • ❌ 広告ネットワーク
  • ❌ ソーシャルメディアトラッキング
  • ❌ あらゆる形式の使用状況トラッキング

6. データセキュリティ

ローカル暗号化

  • API キーは AES-256 暗号化で保存されます
  • 暗号化キーはブラウザに保存されます
  • 誰かがデバイスストレージにアクセスしても、復号化キーなしでは API キーを読み取ることはできません

入力検証

  • すべてのユーザー入力は検証およびサニタイズされます
  • XSS（クロスサイトスクリプティング）攻撃に対する保護
  • HTML コンテンツは自動的にサニタイズされます

7. データの保持と削除

データの保持

すべてのデータはデバイスに保存されるため：

  • データはお客様が能動的に削除するまで保持されます
  • 設定でいつでもすべてのデータをクリアできます
  • 拡張機能をアンインストールすると、すべての ReadLens データが自動的に削除されます

データの削除方法

  1. 個別の記事を削除：記事リストの削除ボタンをクリックします
  2. すべてのデータをクリア：設定 → データ管理 → すべてのデータを削除
  3. 完全削除：Chrome から ReadLens 拡張機能をアンインストールします

8. 児童のプライバシー

ReadLens は13歳未満の児童を対象としていません。私たちは児童から故意に個人データを収集することはありません（実際、誰のデータも収集しません）。

9. プライバシーポリシーの変更

このプライバシーポリシーを変更する場合：

  • このページの上部にある「最終更新日」を更新します
  • 主な変更は拡張機能の更新ノートで発表されます
  • 最新バージョンのプライバシーポリシーはいつでも表示できます

10. お客様の権利

ReadLens はデータを収集しないため、従来のデータ主体の権利（アクセス、訂正、削除など）は適用されません。

しかし、お客様には完全な管理権があります：

  • ✅ いつでもローカルデータを表示、編集、削除できます
  • ✅ いつでもデータをエクスポートできます（JSON、Markdown、CSV 形式）
  • ✅ いつでも AI 機能を無効化または削除できます
  • ✅ API キーを完全に管理できます

このプライバシーポリシーは台湾の法律に準拠します。EU ユーザーについては、GDPR の原則を遵守します（ただし、データを収集しないため、ほとんどの GDPR 要件は適用されません）。

💡 まとめ

ReadLens のプライバシーに対する約束はシンプルです：データはデバイスから出ることがないため、私たちはお客様のデータを見ることができません。

  • ✅ データ収集ゼロ
  • ✅ トラッキングゼロ
  • ✅ サーバーへのアップロードゼロ（サーバーがないため）
  • ✅ 完全なプライバシー保護
