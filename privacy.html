📚 ReadLens 隱私政策

最後更新：2025 年 10 月 11 日

🔒 核心承諾

ReadLens 是完全本地化的應用程式。您的所有資料都儲存在您的裝置上，我們絕不收集、儲存或傳輸您的個人資料。

1. 我們收集什麼資料？

簡短回答：什麼都不收集

ReadLens 不收集任何個人資料。所有資料都儲存在您的本地裝置上。

詳細說明

ReadLens 儲存以下資料僅在您的裝置上：

  • 儲存的文章：您選擇保存的網頁內容
  • AI 摘要：由您的 API Key 生成的摘要（如果啟用）
  • 標籤和筆記：您為文章添加的標籤和個人筆記
  • 設定偏好：您的應用程式設定（主題、語言等）
  • API 金鑰：您的 AI 服務 API Key（加密儲存在本地）

以上所有資料都僅儲存在您的裝置上，ReadLens 開發團隊無法存取。

2. 我們如何使用您的資料？

ReadLens 不使用或存取您的資料，因為：

  • ✅ 無伺服器：沒有後端伺服器收集資料
  • ✅ 無帳號系統：不需要註冊或登入
  • ✅ 無資料上傳：資料永遠不會離開您的裝置
  • ✅ 無第三方追蹤：沒有 Google Analytics 或任何追蹤工具

3. AI 功能與 API Key

🔑 BYOK（Bring Your Own Key）模式

ReadLens 使用「自備金鑰」模式。這意味著：

您的 API Key

  • 加密儲存：API Key 使用 AES-256 加密儲存在您的本地裝置
  • 直接連接：ReadLens 將您的內容直接發送到 AI 提供商（OpenAI、Claude 等）
  • 無中間伺服器：我們沒有中間伺服器，無法看到或記錄您的請求
  • 您的控制：隨時可以刪除或更換 API Key

與 AI 提供商的資料傳輸

當您使用 AI 摘要功能時：

  • 您的文章內容會直接從您的瀏覽器發送到您選擇的 AI 提供商
  • 這些資料不會經過 ReadLens 的伺服器（因為我們沒有伺服器）
  • 請查閱各 AI 提供商的隱私政策：
    • OpenAI 隱私政策
    • Anthropic (Claude) 隱私政策
    • Google (Gemini) 隱私政策

4. Chrome 擴充功能權限

ReadLens 需要以下 Chrome 權限：

activeTab（當前分頁）

用途：讀取您正在瀏覽的網頁內容以保存文章

資料去向：僅儲存在您的本地裝置

storage 和 unlimitedStorage（儲存空間）

用途：在您的裝置上儲存文章、設定和 API Key

資料去向：僅儲存在您的本地 Chrome 儲存空間

contextMenus（右鍵選單）

用途：提供右鍵快速保存功能

資料去向：無資料收集

notifications（通知）

用途：顯示保存成功/失敗的通知

資料去向：無資料收集

https://*/*（網路存取）

用途：從網頁提取內容，以及連接到 AI 提供商 API

資料去向：僅發送到您選擇的 AI 提供商（如果啟用 AI 功能）

5. 第三方服務

ReadLens 本身不使用任何第三方服務。但是：

選用的 AI 服務（BYOK）

如果您選擇啟用 AI 摘要功能並提供自己的 API Key：

  • 您的文章內容會發送到您選擇的 AI 提供商
  • 這些提供商有各自的隱私政策
  • ReadLens 不會看到或記錄這些傳輸

不使用的第三方服務

  • ❌ Google Analytics
  • ❌ 廣告網路
  • ❌ 社群媒體追蹤
  • ❌ 任何形式的使用追蹤

6. 資料安全

本地加密

  • API Key 使用 AES-256 加密儲存
  • 加密金鑰儲存在您的瀏覽器中
  • 即使有人存取您的裝置儲存空間，沒有解密金鑰也無法讀取 API Key

輸入驗證

  • 所有使用者輸入都經過驗證和清理
  • 防止 XSS（跨站腳本）攻擊
  • HTML 內容自動消毒處理

7. 資料保留與刪除

資料保留

由於所有資料都儲存在您的裝置上：

  • 資料會一直保留，直到您主動刪除
  • 您可以隨時在設定中清除所有資料
  • 卸載擴充功能會自動刪除所有 ReadLens 資料

如何刪除資料

  1. 刪除個別文章：在文章列表中點擊刪除按鈕
  2. 清除所有資料：設定 → 資料管理 → 清除所有資料
  3. 完全移除：在 Chrome 中卸載 ReadLens 擴充功能

8. 兒童隱私

ReadLens 不針對 13 歲以下兒童。我們不會有意收集兒童的個人資料（實際上我們不收集任何人的資料）。

9. 隱私政策變更

如果我們變更隱私政策：

  • 會更新此頁面頂部的「最後更新」日期
  • 重大變更會在擴充功能更新說明中通知
  • 您可以隨時查看最新版本的隱私政策

10. 您的權利

由於 ReadLens 不收集任何資料，傳統的資料主體權利（存取、更正、刪除等）並不適用。

但您擁有完全的控制權：

  • ✅ 隨時查看、編輯或刪除您的本地資料
  • ✅ 隨時匯出您的資料（JSON、Markdown、CSV 格式）
  • ✅ 隨時停用或移除 AI 功能
  • ✅ 完全控制您的 API Key
